Chemical

Chemical is an extensible front end web framework.

The source for Chemical is freely available on GitHub at https://github.com/thofmann/chemical.

Anchors are defaulted to a blue color, whether the destination link has been visited or not. The text is underlined when hovered.

Buttons

Buttons come in a variety of flavors.

<button>...</button>
<button class="primary">...</button>
<button class="secondary">...</button>
<button class="success">...</button>
<button class="warning">...</button>
<button class="danger">...</button>

Disabled buttons are styled accordingly.

<button disabled>...</button>
<button class="primary" disabled>...</button>
<button class="secondary" disabled>...</button>
<button class="success" disabled>...</button>
<button class="warning" disabled>...</button>
<button class="danger" disabled>...</button>

Buttons can be easily sized with the 'small' and 'large' classes.

<button class="large">...</button>
<button>...</button>
<button class="small">...</button>

Button styles are also applied to form input buttons.

<form>
<input type="button" value="...">
<input type="submit" value="...">
</form>

Code

The code class can be applied to pre blocks for proper formatting.

npm install stylus -g
        stylus -w src/stylus/chemical.styl -o dist/css/
<pre class="code">...
...</pre>

It can also be used with the span class for inline code formatting.

Here is an example of inline code.

<p>... <span class="code">...</span> ...</p>
